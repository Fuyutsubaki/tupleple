**tupleple タプル操作ライブラリ**


***

C++11準拠のタプル操作ライブラリです

動作確認環境(VS2013)




  aplly - 一般の関数にtupleを適用する

  binary_fold - fold

  cat - tuple_cat

  column - 2次元tupleの列を取り出す zipも参照

  do_nothing - identity 何もしない

  drop - drop

  filter - filter

  map - transform

  move_view - moveのtupleple版

  reverse - reverse

  take - take

  zip - zip columnも参照
